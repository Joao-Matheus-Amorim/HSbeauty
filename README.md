# SalaoApp

Sistema web de agendamento para salões de beleza, estúdios e profissionais autônomos.

## Objetivo

Permitir que clientes agendem serviços online sem precisar instalar aplicativo, enquanto a dona do salão gerencia horários, serviços e agendamentos em um painel administrativo.

## Escopo inicial

- Agendamento online de serviços.
- Cadastro e gestão de serviços.
- Controle de horários disponíveis.
- Painel administrativo da dona do salão.
- Banco de dados relacional.
- Deploy com stack gratuita.

## Stack escolhida

- Front-end: React.js
- Back-end: Node.js + Express
- Banco de dados: PostgreSQL
- Infraestrutura: Docker local, Vercel e Oracle Cloud
- Versionamento: GitHub

## Estrutura inicial

- `frontend/` — interface do cliente e painel admin
- `backend/` — API, regras de negócio e acesso ao banco
- `docs/` — documentação, decisões e roadmap
